# Bidirectional Tutoring for Developmental Motor Learning in Robots: Co-Developed Interaction Dynamics Support Stable Learning

**Authors:** Not specified in the provided text.
**Date:** Not specified in the provided text (arXiv:2606.19728v1).

## Main Findings:

*   **Problem:** Motor skill learning in robots is often treated unidirectionally, overlooking the essential bidirectional nature of social interaction found in human development.
*   **Hypothesis:** Bidirectional tutoring, where both tutor and learner dynamically adapt, leads to more stable and generalized learning by establishing consistent behavioral patterns through co-developed interaction dynamics. Unidirectional tutoring, lacking these constraints, results in less consistent patterns.
*   **Methodology:** Two experiments were conducted using a physical humanoid robot performing an object manipulation task:
    *   One with human-robot interaction.
    *   One with an AI tutor and the robot, using an adaptive intervention mechanism.
    *   A developmental learning framework based on a free-energy-principle-based neural network with generative replay was implemented.
*   **Results:** Across both settings, bidirectional tutoring resulted in:
    *   More consistent robot behaviors.
    *   Stage-wise generalization of learned skills.
    *   Gradual reduction in the need for tutor guidance.
*   **Conclusion:** Bidirectional tutoring serves as an effective, embodied, and socially grounded scaffold for developmental motor learning in robots, highlighting the importance of reciprocal interaction dynamics.
