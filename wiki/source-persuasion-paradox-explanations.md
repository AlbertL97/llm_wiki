## Main content

### Overview
This study investigates how natural-language explanations from LLMs affect objective human-AI team performance and trust calibration across visual reasoning (RAVEN matrices) and logical reasoning (LSAT) (source: The Persuasion Paradox_ When LLM Explanations Fail to Improve Human-AI Team Performance.md).

### The Persuasion Paradox
LLM explanations are highly persuasive and fluent, which decouples user confidence from actual accuracy:
1. **Confidence-Accuracy Decoupling**: Fluent explanations increase user confidence and reliance on the AI, even when the AI's prediction is **incorrect**.
2. **Error Masking**: In visual reasoning, explanations systematically **suppress** users' ability to recover from model errors. Users blindly trust the fluent explanation and override their own correct intuitions.
3. **Alternative Interfaces**: In visual reasoning, interfaces exposing predicted probabilities (uncertainty communication) and using selective automation (deferring uncertain cases to humans) achieve significantly higher team accuracy than explanation-based interfaces.

### Task Dependency & Cognitive Modality
The impact of explanations is strongly mediated by the task's cognitive modality:
- **Visual Reasoning (RAVEN)**: Narrative text explanations degrade performance by creating cognitive mismatch and masking errors.
- **Logical Reasoning (LSAT)**: Narrative text explanations significantly **improve** team accuracy and error recovery, outperforming probability-based indicators. This suggests explanations must match the task's native modality to support effective collaboration (source: The Persuasion Paradox_ When LLM Explanations Fail to Improve Human-AI Team Performance.md).
